# URL Shortener Service

**Category**: API Server

An intelligent URL shortening service with AI-powered link categorization, safety analysis, and click analytics.

## Features (Planned)
- Shorten URLs with custom aliases
- AI-powered URL categorization
- Phishing/malware detection
- Click analytics and geographic data
- AI-generated page summaries
- QR code generation

## Tech Stack
- Cloudflare Workers (edge deployment)
- Cloudflare KV for URL storage
- Workers AI for categorization and safety
- AI Gateway for caching

## Getting Started
npm install
npm run dev
